\section{AI 利用状況と検証(再現性重視)}
\subsection{利用内容}
\begin{itemize}
    \item 利用目的: サーボ制御コードの不具合調査，および作業ログの作成
    \item  入力(プロンプト要旨):
    \begin{itemize}
        \item 「ド」が連続する区間でサーボが動作しない不具合について，原因の調査を依頼．
    \end{itemize}
    \item  出力概要:
    \begin{itemize}
        \item 不具合原因（休符中の音の保持によりサーボが上げっぱなしになる）の特定．
    \end{itemize}
\end{itemize}

\subsection{検証方法}
\begin{itemize}
    \item  検証手法: メロディに対するサーボ動作のシミュレーションおよび実機（結合テスト）での動作確認
    \item  参照資料: サーボ制御コード（servo.ino）およびカエルの歌の楽譜データ
    \item  検証手順:
    \begin{enumerate}
        \item 修正前後のコードで当該区間のサーボ動作をシミュレートし，各音で打鍵（動作）されることを確認．
        \item 実機で結合テストを行い，音・サーボ・LEDマトリクスが連動して動作することを確認．
    \end{enumerate}
\end{itemize}
